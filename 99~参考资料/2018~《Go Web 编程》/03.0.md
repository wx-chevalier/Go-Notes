# 3 Web 基础

学习基于 Web 的编程可能正是你读本书的原因。事实上，如何通过 Go 来编写 Web 应用也是我编写这本书的初衷。前面已经介绍过，Go 目前已经拥有了成熟的 HTTP 处理包，这使得编写能做任何事情的动态 Web 程序易如反掌。在接下来的各章中将要介绍的内容，都是属于 Web 编程的范畴。本章则集中讨论一些与 Web 相关的概念和 Go 如何运行 Web 程序的话题。
